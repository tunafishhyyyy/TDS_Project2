Data Analysis API - File Upload Test

Example Usage

This interface matches the API requirement to accept file uploads, such as:

curl "https://app.example.com/api/" -F "file=@question.txt"
Supported formats: .txt, .csv, .json

Testing with curl

You can also test the API directly with curl:

# File upload
curl "http://localhost:8000/api/" -F "file=@question.txt"

# Form data
curl "http://localhost:8000/api/" -F "task_description=Analyze sales data"

# Check status
curl "http://localhost:8000/api/tasks/{task_id}/status"
